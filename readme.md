# タスクボード

シンプルで使いやすいカンバン方式のタスク管理ツールです。

## 主な機能
- プロジェクトごとのタスク管理
- ドラッグ＆ドロップによる並べ替え（SortableJS）
- 今日の作業計画（Daily Plan）機能
- 作業実績のログ記録と進捗の可視化
- レスポンシブデザイン

## 最近の変更点
- **バグ修正**: タスクを完了表示/非表示にした状態で並べ替えても、順序が崩れないように改善しました。
- **UI改善**: ボードのレイアウトを CSS Grid に変更し、プロジェクトの開閉時に位置がズレないように安定化させました。
- **アニメーション**: プロジェクトの展開・折りたたみにスムーズなスライドアニメーションを追加しました。

## セットアップ
1. PHP が動作する環境（Apache/Nginx等）を用意してください。
2. MySQL/MariaDB でデータベースを作成し、`setup.sql` を実行してください。
3. `config.php` を作成し、DB接続情報を設定してください。
4. ブラウザで `index.php` にアクセスします。
